# phpCaptcha

php simple Captcha

#### Requerimientos

* [lamp](https://bitnami.com/stack/lamp/installer)
* [nodejs](https://nodejs.org/es/)
* [npm](https://www.npmjs.com/)


#### Instalacion

```

npm i phpCaptcha

```
#### Configuracion

Se pueden configurar 
* colores
* largo Captcha
* combinacion de caracteres a utilizar
* letra a utilizar

Esto se puede hacer desde el archivo:

```

/js/config.js

```
